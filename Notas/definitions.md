# Definiciones Preacticas

## Variables
Una variable es un espacio de memoria donde se almacena un dato que puede cambiar durante la ejecución de un programa.

nombre = "Juan"
edad = 20

## Listas
Una lista es una estructura de datos que permite almacenar varios valores en un solo lugar y mantenerlos ordenados.

frutas = ["manzana", "pera", "uva"]

Características:

- Puede almacenar varios elementos.
- Tiene posiciones llamadas índices.
- Puede modificarse (agregar, eliminar o cambiar elementos).

## Diccionario
Un diccionario es una estructura de datos que almacena información mediante pares de clave y valor.

persona = {
    "nombre": "Carlos",
    "edad": 25,
    "ciudad": "Bogotá"
}
